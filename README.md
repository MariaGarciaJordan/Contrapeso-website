# Contrapeso Website

Landing page en React para Contrapeso.

## Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- three.js
- @react-three/fiber
- @react-three/drei
- Vitest
- React Testing Library
- Playwright

## Objetivo

Construir una landing moderna, dinámica y alineada con el manual de marca de Contrapeso.

## Estructura

- `public/assets` recursos estáticos
- `src/components` componentes reutilizables
- `src/sections` bloques de página
- `src/lib` utilidades
- `src/styles` estilos auxiliares
- `src/test` setup de testing
- `e2e` pruebas end-to-end

## Scripts

- `npm run dev`
- `npm run build`
- `npm run preview`
- `npm run lint`
- `npm run typecheck`
- `npm run test`
- `npm run test:e2e`
- `npm run qa`
